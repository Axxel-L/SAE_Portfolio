Projets et exercices réalisés en cours · BUT2 Informatique

Tableau 3D · WebGL

Three.js · WebGL

Reproduction d'un tableau réel en scène 3D interactive sur le web avec Three.js. Très sympa visuellement mais complexe : placement des éléments en 2D délicat, gestion de la skybox fastidieuse, ombres capricieuses et textures qui buguent par moments.

WikiCoaster

Symfony · PHP

Site web réalisé avec le framework Symfony. Beaucoup de concepts à maîtriser au début (routes, contrôleurs, entités, migrations, Twig) mais une fois qu'on a compris la logique, le framework devient incroyablement puissant et productif.

Interfaces Graphiques 3D

Three.js · WebGL

Création d'interfaces graphiques en 3D avec Three.js. Très fun à réaliser mais la complexité est réelle : indispensable de jongler entre documentations, exemples et IA pour s'en sortir sur les calculs de position, les lumières et les caméras.

Application iOS · Swift

Swift · SwiftUI

Création d'une application native iOS avec Swift et SwiftUI. Expérience incroyable : concevoir sa propre app et la tester directement sur son iPhone en temps réel. SwiftUI rend la création d'interfaces étonnamment facile et intuitive.

API Toutaix

Docker · API REST

Déploiement progressif d'une API en vue d'un projet noté. Assez simple à gérer grâce à Docker : chaque modification se déploie facilement, les conteneurs isolent les services et permettent des mises à jour rapides sans casser l'existant.

Application Flutter

Flutter · Dart

Création d'une application mobile cross-platform avec Flutter. Intéressant et puissant, mais le langage Dart a une courbe d'apprentissage raide et Gradle peut poser problème si les versions du SDK Android et de Flutter ne sont pas parfaitement alignées.

API · Database First

Docker · API

Création d'une API simple en approche database first. Facile à mettre en place et à maintenir grâce à Docker : le schéma de base pilote la génération du code, les conteneurs assurent la cohérence de l'environnement entre les membres de l'équipe.
